Effects SDK 文档

SenseAR_v8.7.4（Effects）正式发版

SenseAR_Effects SDK为直播、短视频等泛娱乐化产业提供特效的解决方案，包含人脸检测及追踪、属性检测，前后背景分割，手势检测及追踪，美颜、美妆、美体塑性、2D贴纸、3D特效、脸部变形，通用物体跟踪，滤镜等功能。

SenseAR_Effects SDK拥有市场领先的人脸关键点检测及追踪技术：稳定性强，跟踪平稳，噪音抖动少；具备出色的鲁棒性：完美支持大角度，大表情，极端光照，场景抖动，大小等变化。SenseAR_Effects SDK可毫秒级检测并追踪人脸关键点。采用 GPU处理，使得人脸检测追踪、美颜、贴纸、滤镜等功能处理速度更快，可以达到在市场主流机型720P 实时流畅预览的效果。

SDK重要更新：

贴纸功能更新：

o 新增试鞋特效，支持自主制作试鞋素材。

o 新增猫脸3D特效，并且支持自由更换遮挡体，支持自主制作猫脸3D素材。

o 新增狗脸3D特效，并且支持自由更换遮挡体，支持自主制作狗脸3D素材。

TRYON功能更新：

o TRYON试妆新增唇线、眼影、眼线、眼印、眼睫毛、眉毛、腮红、修容、粉底。

主要功能：

• 人脸检测及跟踪：毫秒级人脸检测和追踪，准确的使用106关键点对人脸信息进行描述。

• 人脸动作检测：眨眼，张闭嘴，点头，摇头，抬眉毛、嘟嘴、笑脸等脸部动作检测，在实际使用场景中可通过脸部动作触发贴纸效果。

• 手势检测及跟踪：可检测并跟踪手掌、大拇指、抱拳作揖、托手、爱心、单手比心、OK，Yeah，手枪，手指，3共计11种手势动作。在实际使用场景中可通过手势触发贴纸效果。

• 人脸属性检测：可以检测数当前人脸的年龄、性别、颜值三个属性值。

• 前后背景分割：在图像中检测人物与背景，通过识别出人物和背景的关系，将人物和背景分开，同时可添加人物描边效果。

• 通用物体跟踪：增加通用物体跟踪功能。可实时跟踪视频中的指定的区域。跟踪区域大小可变，支持高清实时跟踪，处理速度快。

• 人脸动态2D贴纸特效：基于人脸关键点检测跟踪结果，通过2D渲染引擎将二维素材实时的渲染到人脸的指定位置，支持脸部实时跟踪和脸部大角度贴合，支持声音效果。

• 人脸动态3D贴纸特效：基于人脸关键点检测跟踪结果，通过3D渲染引擎实时得渲染3D材质效果和灯光效果，支持跟随人头进行平移，转动等动作，支持漫反射、高光、环境反射、自发光、凹凸、预烘焙等材质效果。

• 粒子特效：可以真实、绚丽得模拟雪、尘、火、爆炸、流星尾迹等抽象视觉效果。支持设置粒子发射速度、形状、颜色、角速度等模块。粒子效果可以支持由动作触发；可以跟随人脸关键点、手部移动。目前可以支持5种脸部动作和10种手部动作触发。

• Avatar功能：使用人脸检测功能对用户面部进行重建，通过感应用户面部表情变化，实时驱动3D动画模型做出与用户面部表情一致的动作表情。

• 美颜：磨皮、美白、红润、对比度、饱和度、锐化、清晰、大眼、瘦脸、小脸、窄脸、圆眼、共计12种美颜效果。

• 微整形：可对面部五官进行精细化调整，让五官更显精致。发际线调节、鼻子长短调节、鼻翼调节、人中调节、嘴型调节、下巴调节、瘦脸型调节、侧脸隆鼻、开眼角、开外眼角、亮眼、苹果肌、去法令纹、去黑眼圈、牙齿美白、眼睛角度、眼间距、瘦颧骨，小头等功能。

• 脸部变形：目前支持两种方案进行脸部变形，可呈现外星人，哈哈镜、丑妆等效果。

• 猫脸检测功能：可在图像中实时检测猫脸信息。

• 猫脸贴纸功能：支持对检测到的猫脸增加2D贴纸功能，支持人脸和猫脸贴纸同时渲染，支持人脸和猫脸穿戴不同的贴纸。

• 天空盒：使用户置身于虚拟环境的包围中。支持与前后背景分割功能同时使用。

• 变脸美妆功能：通过人脸关键点信息绘制人脸三维面片的方式，将二维纹理立体的绘制在人脸上，能够极大的贴合人脸各个角度，达到立体逼真的视觉效果。在细节效果的呈现上，可以跟随人脸五官进行实时位置变化。支持染发、眉毛，眼影，眼线，睫毛，腮红，口红，修容，美瞳等效果强度调节及功能叠加。

• 滤镜：111种滤镜选择，满足不同场景的各种需求。

• 支持各个功能/模块在同一个素材包中使用（2D贴纸，前后背景分割，手势贴纸，脸部变形，变脸美妆，3D贴纸，粒子功能，声音模块，滤镜功能，美颜功能）。
