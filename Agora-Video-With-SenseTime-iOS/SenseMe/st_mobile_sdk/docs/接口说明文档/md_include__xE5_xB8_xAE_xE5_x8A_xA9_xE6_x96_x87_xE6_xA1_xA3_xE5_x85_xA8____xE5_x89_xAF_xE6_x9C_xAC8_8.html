1.人脸106点检测及追踪

人脸106点检测及追踪介绍

人脸关键点检测是指在检测到的人脸框中，进一步定位人脸的五官和轮廓位置。在图片给定的人脸框中对关键点的坐标进行迭代收敛，返回迭代的结果。若框中没有人脸，依然会返回结果。人脸关键点跟踪是在检测的基础上，加入面部轨迹追踪与视频动态解析技术，极大地提高了人脸的检测速度，能够随着视频内容的变化迅速定位人脸所在的位置。

人脸关键点跟踪技术，采用了最新的基于级联回归的算法，结合了深度学习的五官标定初始化，综合多个不同标准的多点数据集知识，使得同一个模型可以应用于不同数量的关键点跟踪，以保障更低的误差和更好的适应性。我们的人脸关键点定位算法可对各种表情、姿态、角度丰富多变的人脸进行精准关键点定位。

产品功能：

  1. 静态图片中人脸106关键点检测功能
  2. 动态视频中人脸106关键点的实时检测和跟踪功能
  3. 支持最多32个人脸的多人脸实时检测和跟踪
  4. 支持实时人脸的三维旋转角度输出
  5. 可检测张嘴、点头、摇头、挑眉、眨眼、笑、嘟嘴共7个脸部动作
  6. 支持360人脸检测（使用360度人脸检测模型）

产品优势：

  • 支持人脸大角度变化场景
  • 支持人脸大表情变化场景
  • 支持人脸大小变化场景
  • 支持人脸遮挡场景
  • 支持极端光照变化场景
  • 支持场景抖动
  • 支持对面部遮挡的检测，输出106点遮挡信息

3. 3Dmesh

3.1 3Dmesh功能介绍

3Dmesh是指在检测到的人脸框中提供高精度人脸稠密点，能够完美贴合人脸并清晰描述人脸三维曲面信息。在图片给定的人脸框中对人脸稠密点进行回归，并返回人脸稠密点坐标、拓扑面片、法向量和三维空间变换矩阵。若框中没有人脸，依然会返回结果。

3Dmesh技术采用了轻量级深度神经网络并使用多种真实场景下的三维人脸数据进行端到端学习，使其能在手机端实时运行并保障人脸稠密点效果的鲁棒性。我们的3Dmesh算法可对各种表情、姿态、角度丰富多变的人脸进行精准贴合，支持制作精良的美妆或贴纸特效。目前需要依赖106或240关键点使用。

4. 360°HeadMesh

4.1 360°HeadMesh功能介绍

360°HeadMesh是在搭配轻量级跟踪模型的基础上，在人头360度Yaw旋转场景能够检测到人头框并提供高精度人头稠密点，输出人头曲面信息。在图像中检测人头框并对整个人头稠密点进行回归，返回人头稠密点坐标、拓扑面片和三维空间的变换矩阵。该算法能在手机端实时、准确、高鲁棒性的运行，能够贴合各种表情、姿态多变的人脸，支持制作正脸以及背面的美妆和贴纸特效。

10. 人脸比对功能

SenseAR人脸比对SDK能够快速、准确地将目标人脸与人脸比对库中的人脸进行比对，达到精准实现人脸比对，身份识别的目的。
人脸比对SDK通过人脸检测，提取人脸特征值，将该特征值与目标库(特征值目标库)进行比对，即可输出比对相似度。
可通过设置比对相似度阈值，在相似值达到阈值后返回人脸比对的结果。

33. 深度估计

深度估计是指映射图像中每个像素点为其相对拍摄源的距离，从而得到稠密的相对或绝对深度图。深度估计可以提供场景中目标的距离、轮廓等三维结构信息，可应用于三维重建、相机拍摄景深效果以及其他AR场景应用。
