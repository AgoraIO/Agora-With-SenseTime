设置跟踪目标的矩形

参数
[in]handle已初始化的实时通用物体跟踪句柄
[in]image用于检测的图像数据
[in]pixel_format用于检测的图像数据的像素格式,都支持,不推荐BGRA/BGR/RGBA,速度会慢,内部会统一将图片covert to gray
[in]image_width用于检测的图像的宽度(以像素为单位)
[in]image_height用于检测的图像的高度(以像素为单位)
[in]image_stride用于检测的图像的跨度(以像素为单位),即每行的字节数;目前仅支持字节对齐的padding,不支持roi
[in]target_rect输入指定的目标矩形框,输出实际跟踪的矩形框.目前只能跟踪2^n正方形矩形,
返回
成功返回ST_OK,失败返回其他错误码,错误码定义在st_mobile_common.h中,如ST_E_FAIL等

)

#include <st_mobile_object.h>

对连续视频帧中的目标进行实时快速跟踪

参数
[in]handle已初始化的实时通用物体跟踪句柄
[in]image用于检测的图像数据
[in]pixel_format用于检测的图像数据的像素格式,都支持,不推荐BGRA/BGR/RGBA,速度会慢,内部会统一将图片covert to gray
[in]image_width用于检测的图像的宽度(以像素为单位)
[in]image_height用于检测的图像的高度(以像素为单位)
[in]image_stride用于检测的图像的跨度(以像素为单位),即每行的字节数;目前仅支持字节对齐的padding,不支持roi
[out]rects输入指定的目标矩形框,输出实际跟踪的矩形框.目前只能跟踪2^n正方形矩形,如果追踪失败则为{0, 0, 0, 0},如果框有一条边移出图片太远会被认为检测失败，这个距离阈值是框边长的0.6倍
[out]score置信度，取值范围(0,1)
返回
成功返回ST_OK,失败返回其他错误码,错误码定义在st_mobile_common.h中,如ST_E_FAIL等
