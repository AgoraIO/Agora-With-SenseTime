◆ st_mobile_avatar_get_expression()

ST_SDK_API st_result_t st_mobile_avatar_get_expression ( st_handle_t  handle,
int  width,
int  height,
st_rotate_type  rotate,
const st_mobile_face_t *  p_face,
float *  p_expression_array 
)

获取根据传入关键点拟合的Avatar表情系数结果，非线程安全

参数
[in]handle已初始化的Avatar句柄
[in]width预览图像宽度
[in]height预览图像高度
[in]rotate预览图像中将人脸转正需要的旋转角度
[in]p_face当前帧人脸关键点检测结果。目前SDK只支持对一个人脸获取表情系数
[out]p_expression_array待写入的参数数组指针，数组应由调用方预先分配，数组大小应该大于或等于ST_AVATAR_EXPRESSION_NUM，否则将产生越界异常
返回
成功返回ST_OK,失败返回其他错误码,错误码定义在st_mobile_common.h中,如ST_E_FAIL等
